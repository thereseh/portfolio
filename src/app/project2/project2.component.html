Food & Beer

A web application utilizing external APIs to acquire data

I used Yummly and BreweryDB API to create an application that returns recipes and beer. You can either do a random search, or input specific values.

I'm currently working on a continuation of this app that will allow the user to store recipes using POST/PUT/DELETE Ajax call to an API, so come back soon to check it out!
