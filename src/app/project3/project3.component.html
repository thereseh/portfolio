Pictionary

A basic pictionary game created in order to learn how to combine multi-user functionallity and HTML5 canvas

I'm allowing clients to interact with each other using Socket.io to send information back and forth from the server. A simple text file is loaded and used to randomly get objects to draw.

I'm planning to add more functionality, such as colors and stroke-size to this game once I have some spare time!
