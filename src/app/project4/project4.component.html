Battleship

A web application utilizing socket.io to allow clients to play against each other in real time.

I set up the server for this project using NodeJS. Data is passed between clientes using socket.io.

The first two people that joins the server will become the 2 players of the game. All the others who join will become spectators that will see the two canvases update in real time as each player makes a move.

This application is currently under construction, as all functionallity is not yet implemented, and I need to polish the design.
