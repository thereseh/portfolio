Asteroids

A basic 2D Asteroid inspired game with my own take on the design.

The bee is your vehicle, and the flowers are the asteroids. You get the idea! This was a class project that I developed in order to learn how to calculate my own velocity and direction vectors, and also how to calculate collision. In this case, I'm using Circle Detection to achieve this. Specifically, I'm calculating the distance between objects, if it's less than the sum of their CircleCollider component, then they are colliding.

Gameplay: move around the screen using the arrow keys, and shoot with the space bar.

When a flower is hit, it will break up into 1 to 3 smaller pieces. You have 3 lives, and will be "cloaked" for a few seconds after taken damage, which allows you to gain control of the situation.
