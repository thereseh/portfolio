Portfolio

Designed from the ground up using Angular2

When it was time for me to build this site I decided to embrace the opporturnity to learn a new framwork. After some consideration, I went for Angular2.

In addition, Bootstrap is used to help with the design. For example, the project page is using the carousel component to wrap the project images. It allowed me to easily format the images and put information text over them.

I also wrote a NodeJS express server, which was necessary for Heroku to be able to build and deploy the project.
